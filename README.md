## Hasil Akhir

> Untuk hasil akhir saya hanya memasukkan beberapa saja :)

<img src="./public/img/Hasil Akhir - 1.png" width="600" align="center" alt="GET book api">

Gambar Hasil Akhir - 1:

dengan menggunakan method GET untuk menampilkan semua data book.



<img src="./public/img/Hasil Akhir - 2.png" width="600" align="center" alt="GET book api">

Gambar Hasil Akhir - 2:

dengan menggunakan method GET untuk menampilkan 1 data book. Anda bisa mengganti **{{masukkan_id_disini}}** menjadi ID yang dituju berupa angka atau integer (seperti pada contoh menggunakan angka 1).



<img src="./public/img/Hasil Akhir - 3.png" width="600" align="center" alt="GET book api">

Gambar Hasil Akhir - 3:

dengan menggunakan method POST dan Body x-www-form-form-urlencode untuk menambah data book. Anda bisa mengganti **VALUE** dari **KEY** sesuai kebutuhan.



<img src="./public/img/Hasil Akhir - 4.png" width="600" align="center" alt="GET book api">

Gambar Hasil Akhir - 4:

dengan menggunakan method PUT dan Body x-www-form-form-urlencode untuk mengubah data book. Anda bisa mengganti **{{masukkan_id_disini}}** menjadi ID yang dituju berupa angka atau integer (seperti pada contoh menggunakan angka 4).



<img src="./public/img/Hasil Akhir - 5.png" width="600" align="center" alt="GET book api">

dengan menggunakan method DELETE untuk menghapus satu data book. Anda bisa mengganti **{{masukkan_id_disini}}** menjadi ID yang dituju berupa angka atau integer (seperti pada contoh menggunakan angka 4).
